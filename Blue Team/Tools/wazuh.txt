Wazuh
- An open-source security monitoring platform that provides threat detection, log analysis, intrusion detection, and endpoint security monitoring.

==================================================
1. Overview
==================================================
Wazuh is an open-source security platform used for **endpoint detection, log analysis, and threat monitoring**.

It provides capabilities similar to an **Extended Detection and Response (XDR)** platform by collecting and analyzing security data from endpoints, servers, and cloud environments.

Wazuh builds on the **OSSEC host-based intrusion detection system (HIDS)** and integrates with the **Elastic Stack** for visualization and analysis.

It is commonly used for:

- endpoint monitoring
- threat detection
- log analysis
- security compliance monitoring
- incident investigation

Wazuh agents are installed on endpoints to collect security data and forward it to a centralized server.
