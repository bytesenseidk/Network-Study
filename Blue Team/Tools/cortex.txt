Cortex
- An open-source security automation and analysis engine used to enrich indicators and automate investigations during incident response.

==================================================
1. Overview
==================================================
Cortex is a security automation platform developed by TheHive Project.  
It allows security teams to analyze indicators of compromise (IOCs) and automate security investigations.

Cortex integrates with many external security services to perform automated analysis such as:

- malware analysis
- IP reputation lookup
- domain reputation analysis
- file sandboxing
- threat intelligence enrichment

Cortex is commonly used alongside **TheHive** incident response platform, where analysts can launch automated investigations directly from incident cases.

The primary goal of Cortex is to **reduce manual investigation work and speed up security analysis**.

5. Example Investigation Workflow
==================================================

Step 1 — Security Alert Triggered

A SIEM or monitoring tool generates a security alert.


Step 2 — Indicator Extraction

Suspicious indicators are identified, such as:

- IP addresses
- domains
- file hashes


Step 3 — Submit Indicator to Cortex

The analyst sends the indicator to Cortex for analysis.


Step 4 — Automated Analysis

Cortex queries external threat intelligence sources.


Step 5 — Results Returned

Cortex provides reputation scores, threat intelligence, and analysis results.


Step 6 — Investigation Continues

The analyst uses the results to determine whether the indicator is malicious.
